# 🚀 How to Use Your Installed Antigravity Skills

You currently have a massive library of over **300+ Antigravity Skills** installed on your machine (`~/.gemini/antigravity/skills/`). 

Skills are specialized instruction sets, architectures, and tools that I (Antigravity) can load into my context on demand to become an expert in a specific domain. 

## 🛠️ How to Trigger a Skill

Because there are so many skills, I don't load them all into memory at once. To use a skill, **you just need to explicitly mention it or describe the task you want me to do.** I will automatically find the matching skill, read its `SKILL.md` file, and apply its instructions to your request.

**Examples of how to prompt me:**
- *"Use the `supabase-automation` skill to..."*
- *"Act as a `backend-architect` and analyze this codebase..."*
- *"Can you run an SEO audit on our landing page? (Use the `seo-audit` skill)"*
- *"I need you to build a complex UI. Please load `ui-ux-pro-max` and `tailwind-design-system`."*
